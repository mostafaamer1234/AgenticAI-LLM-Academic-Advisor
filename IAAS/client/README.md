
# Intelligent Academic Advising System (React)

## Setup Instructions

1. Unzip the folder.
2. Run:
    ```
    npm install
    npm start
    ```

3. Open [http://localhost:3000](http://localhost:3000) to see the app.

## Features
- Login page
- Dashboard with Course Planner and AI Chat
- React Router for navigation
- Plain CSS for styling
